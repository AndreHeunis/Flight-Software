Transmit and/or receive a buffer of data over the specified I2C channel.

Author
Pieter J. Botma
Date
23/05/2012

This function starts a communication sequence, of type flag, with a slave module, address, by transmitting txBuffer and or receiving rxBuffer over the specified I2C channel.

Parameters
[in]i2cPointer to the I2C module to be used.
[in]addressI2C addres of slave to initiate communication with.
[in]flagIndicates the type of communication sequence.
[in]txBufferPointer to data buffer to be transmitted over the specified I2C channel.
[in]txBufferSizeSize of data buffer to be tranmistted.
[out]rxBufferPointer to buffer where data, received over the specified I2C channel, should be saved.
[in]rxBufferSizeNumber of bytes received over the specified I2C channel.
