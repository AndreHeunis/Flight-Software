void BSP_RTC_tick ( )

Increment RTC in milliseconds.

Author
Pieter J. Botma
Date
14/05/2012

This function increments the milliseconds counter of the RTC. If 1024 ticks have been reached (1024*32 = 32768 = LFXO), the seconds counter is incremented and the milliseconds counter is reset.
